# Seedance 2.0 Prompt Skill for Claude Code

[English](README.md)

> 让 Claude Code 成为你的 Seedance 2.0 视频提示词专家

一个为 [Claude Code](https://docs.anthropic.com/en/docs/claude-code) 打造的 Skill，专门生成可直接粘贴到字节跳动[即梦平台](https://jimeng.jianying.com/) Seedance 2.0 使用的高质量中文视频提示词。

基于 **400+ 条 Seedance 2.0 实战提示词**提炼而成，涵盖分镜时间轴法、强化词约束体系、18 大场景写作指南等核心方法论。

## 特性

- **十大创作模式** — 纯文本生成、角色一致性、运镜复刻、特效模仿、剧情补全、视频延长、声音设计、一镜到底、视频编辑、音乐卡点
- **强化词约束体系** — 从实战提示词中提炼的质量保险机制，确保生成稳定性（画质、人物稳定、动作流畅、光影、氛围）
- **分镜时间轴法** — 按时间段精确控制景别、镜头运动和画面内容，4-15 秒灵活适配
- **18+ 场景写作指南** — 每类场景都有专属的写作重心、强化词配方和典型风格参考
- **超长视频分段策略** — 超过 15 秒的视频自动拆分为多段，含衔接点描述
- **多模态素材引用** — 完整的 `@图片`/`@视频`/`@音频` 引用规范
- **写作八维度** — 面部、动作、光影、材质、镜头、转场、物理、声效的精细化写作标准

## 快速开始

### 1. 安装 Skill

将 `seedance2.md` 放入你的 Claude Code skill 目录：

```bash
# 方式一：放入项目级 skill 目录
mkdir -p .claude/skills
cp seedance2.md .claude/skills/

# 方式二：放入全局 skill 目录（所有项目可用）
mkdir -p ~/.claude/skills
cp seedance2.md ~/.claude/skills/
```

### 2. 使用

在 Claude Code 中直接描述你想要的视频，Skill 会自动触发：

```
> 帮我写一个赛博朋克风格的未来城市视频提示词

> 生成一段15秒的京剧花旦表演视频prompt

> 写一个口红带货视频的提示词，9:16竖屏

> 我需要一个30秒的仙侠战斗短片
```

**触发关键词**：`seedance`、`sd2`、`视频提示词`、`video prompt`、`生成视频`、`AI视频`、`即梦`、`短剧`、`广告视频`、`视频延长`、`带货视频` 等。

## 核心方法论

### 黄金公式

```
主体 + 动作 + 场景 + 光影 + 镜头语言 + 风格 + 画质 + 【强化词】约束
```

### 分镜时间轴

| 时间段 | 景别 | 叙事功能 |
|--------|------|----------|
| 0-3秒 | 全景/大远景 | 建立世界观 |
| 3-7秒 | 中景 | 主体登场、叙事推进 |
| 7-11秒 | 近景/特写 | 情感爆点、核心表现 |
| 11-15秒 | 特写↔全景 | 收束升华、意境留白 |

### 强化词体系

每条提示词结尾的质量约束，用 `【强化词】` 标记：

```
【强化词】4K超高清，电影质感，面部清晰稳定，五官全程一致不变形，
人体比例自然协调，动作流畅连贯无跳帧，暖黄逆光效果，景深感强。
```
